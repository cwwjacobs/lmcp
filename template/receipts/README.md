# Receipts

This folder records pack build, index, delivery, and validation receipts.

Expected receipts:

- `PACK_RECEIPT.md` — human-readable pack receipt.
- `index-build-<timestamp>.json` — machine-readable index build receipt.
- `delivery-<customer-or-pack>.md` — optional customer delivery receipt.

Receipts are evidence. They do not grant authority or prove safety.
