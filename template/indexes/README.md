# Generated indexes

This folder is generated by `scripts/build_lmcp_index.py`.

Expected files:

- `hash-ledger.jsonl` — one record per discovered skill folder, keyed by content hash.
- `binding-index.jsonl` — cheap machine binding markers for routing.
- `registry.jsonl` — promoted skill records for selection.
- `tag-index.json` — inverted index of tag to skill IDs.

Do not hand-edit generated indexes unless creating a deliberate fixture.
